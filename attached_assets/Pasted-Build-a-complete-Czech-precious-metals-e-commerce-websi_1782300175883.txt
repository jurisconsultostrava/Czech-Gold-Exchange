Build a complete Czech precious metals e-commerce website called SwissGold.cz deployed on Railway.
Design
See in DESIGN SYSTEM
Live Price Feed
Primary price source: https://xaumanager.cz/api/export/xml?hash=moje-zlato-secret
XML structure:
<SHOPITEM>
  <CODE>30070</CODE>
  <NAME>1 Kilo Gold Bar | Argor-Heraeus</NAME>
  <PRICE_VAT>2852701</PRICE_VAT>
  <PURCHASE_PRICE>2838508</PURCHASE_PRICE>
  <STOCK><AMOUNT>1</AMOUNT></STOCK>
  <AVAILABILITY_IN_STOCK>Skladem</AVAILABILITY_IN_STOCK>
</SHOPITEM>

PRICE_VAT and PURCHASE_PRICE are in haléře (divide by 100 to get Kč). Fetch via server-side API route /api/prices every 60 seconds to avoid CORS. Cache in memory.
Spot prices: https://xaumanager.cz/api/public/spot Response: {spots: [{metal, price_per_gram_czk, price_per_oz_czk}], eur_czk, ts} Use for live ticker.
Database Schema (Prisma)
model Product {
  id           String   @id
  name         String
  manufacturer String?
  weightGrams  Float
  fineness     String   @default("999.9")
  category     String
  subcat       String
  year         Int?
  featured     Boolean  @default(false)
  active       Boolean  @default(true)
  image        String?
  description  String?
  sortOrder    Int      @default(0)
  createdAt    DateTime @default(now())
  updatedAt    DateTime @updatedAt
}

model Order {
  id              String      @id @default(cuid())
  orderNumber     String      @unique
  status          String      @default("new")
  customerName    String
  customerEmail   String
  customerPhone   String?
  customerAddress String?
  customerCity    String?
  customerZip     String?
  paymentMethod   String      @default("bank_transfer")
  deliveryMethod  String      @default("personal")
  totalCzk        Float
  totalEur        Float
  currency        String      @default("CZK")
  createdAt       DateTime    @default(now())
  items           OrderItem[]
}

model OrderItem {
  id           String  @id @default(cuid())
  orderId      String
  order        Order   @relation(fields: [orderId], references: [id])
  productId    String
  productName  String
  weightGrams  Float?
  quantity     Int     @default(1)
  unitPriceCzk Float
  unitPriceEur Float
}

model BuybackRequest {
  id              String   @id @default(cuid())
  requestNumber   String   @unique
  status          String   @default("new")
  customerName    String
  customerEmail   String
  customerPhone   String?
  itemDescription String?
  estimatedCzk    Float?
  adminNote       String?
  createdAt       DateTime @default(now())
}

model Settings {
  id                    Int   @id @default(1)
  eurToCzk              Float @default(25.20)
  globalMarginCzk       Float @default(300)
  marginInvestZlato     Float @default(300)

  bulkTier2DiscountPct  Float @default(0.1)

model PriceOverride {

Product Naming Convention (Czech)
Bar:  {Manufacturer} zlatý/stříbrný/platinový/palladiový slitek {weight} [{series}]
Coin: {Mint} zlatá/stříbrná/platinová/palladiová mince {series} {year} {weight}

Examples:
"Argor-Heraeus zlatý slitek 1 kg"
"Valcambi zlatý slitek 1 kg ražený"
"PAMP Suisse zlatý slitek 100 g litý"
"Royal Canadian Mint zlatá mince Maple Leaf 2026 1 oz"
"Münze Österreich zlatá mince Wiener Philharmoniker 2026 1 oz"
"South African Mint zlatá mince Krugerrand 2026 1 oz"
"U.S. Mint zlatá mince American Eagle 2026 1 oz"
"Perth Mint zlatá mince Kangaroo 2026 1 oz"

Seed Script
Fetch XML from https://xaumanager.cz/api/export/xml?hash=moje-zlato-secret, parse all 215 SHOPITEMs, for each:
. Párování produktů probíhá podle CODE nebo ITEM ID
·	Detect metal (gold/silver/platinum/palladium) from name
·	Detect kind (bar/coin) from name keywords
·	Detect manufacturer/mint from name
·	Detect weight from name
·	Detect year from name
·	Build Czech name using naming convention above
·	Set category and subcat
·	Set image = https://goldspot.cz/obrazky/{CODE}/1.webp
·	Insert into products table

Správa přirážky (marže)
V admin panelu záložka "Cenotvorba":
·	Globální přirážka v Kč (přičtena ke každé ceně z XML feeds) — default 300 Kč
·	Přirážka per kategorie (investicni-zlato, investicni-stribro, platina-palladium, mince-cnb) — každá kategorie má vlastní přirážku
·	Přirážka per produkt (override pro konkrétní CODE) — tabulka PriceOverride
·	Výkupní spread v % (kolik % pod PURCHASE_PRICE z XML se zobrazí zákazníkovi) — default 0 %
·	Sleva pro terminované dodání v % — default 9 %
·	Množstevní slevy — editovatelné prahy (default: 2 ks = -0.05%, 5 ks = -0.1%, 10 ks = -0.2%)
Všechny hodnoty uložené v tabulce Settings, editovatelné v reálném čase.
prisma
model PriceOverride {
  productId     String @id
  marginCzk     Float?   // pevná přirážka v Kč pro tento produkt
  marginPct     Float?   // nebo % přirážka
  active        Boolean @default(true)
}
Import / Export (CSV a XML)
V admin panelu záložka "Import / Export":
Export:
·	Export XML — vygeneruj Shoptet-kompatibilní XML feed všech aktivních produktů s aktuálními cenami z xaumanager XML + naše přirážka. URL: /api/admin/export/xml
·	Export CSV — stejná data ve formátu CSV (code, czech_name, price_czk, purchase_price_czk, category, manufacturer, weight, availability). URL: /api/admin/export/csv
·	Tlačítka ke stažení přímo v UI
Import:
·	Import XML — nahrát Shoptet XML soubor, systém porovná CODE, aktualizuje: name (czech), category, subcat, manufacturer, weight, fineness, year, description, active. Nové produkty přidá, existující aktualizuje.
·	Import CSV — stejné jako XML ale z CSV souboru (sloupce: code, name, manufacturer, category, subcat, weight_grams, fineness, year, description, featured, active)
·	Po importu zobrazí report: X přidáno, Y aktualizováno, Z přeskočeno (neznámý CODE)
·	Validace před importem — zobrazit náhled prvních 10 řádků před potvrzením
